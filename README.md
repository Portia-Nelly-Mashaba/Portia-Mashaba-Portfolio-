## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/Portia-Nelly-Mashaba/Portia-Mashaba-Portfolio-.git

### Installation

Follow these steps:

```sh
# Step 1: Clone the repository
git clone https://github.com/Portia-Nelly-Mashaba/Portia-Mashaba-Portfolio-.git

# Step 2: Navigate to the project directory
cd smart-portfolio-ai-dashboard

# Step 3: Install the necessary dependencies
npm install

# Step 4: Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`

## Development

```sh
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run linter
npm run lint
```
